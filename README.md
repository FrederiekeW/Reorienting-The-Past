# Reorienting the Past - On the Perspective, Function, and Affordances of Portable Magdalenian Art
Code for my Cognitive Science Master's Thesis on Portable Magdalenian Objects

#### **Title:** Reorienting the Past - On the Perspective, Function, and Affordances of Portable Magdalenian Art
#### **Author:** Frederieke Nicola Wullf
#### **Year:** 2026


### **Description:**
This repository contains my analysis code for my master's thesis in Cognitive Science research, which is based on using eye tracking, to investigate the visual interaction with portable Magdalenian objects. 

The code is primarily written in R, and covers the overall analytical pipeline: From pre-processing, statistical modelling, and creating visualisations for the thesis. 


The repository is organised as follows:

**Analysis_H1a_H1b_H2_H3**: Statistical analysis pipeline for model 1, testing hypotheses 1a, 1b, 2, and 3.

**Analysis_H4**: Statistical analysis pipeline for model 2, testing hypothesis 4.

**Model_Comparison**: LOO-CV based model comparisons to select the best fitting models.

**Plots**: Visualisation scripts and descriptive results.

**Pre-processing**: Pre-processing pipeline for cleaning the raw eye tracking data.

**saliency_maps**: Generating saliency maps and saliency map overlays for figure 1.





*Raw data is not included in this repository due to participant confidentiality*
